Eventzilla Checkout API Sample Application

This sample application demonstrates the usage of API v2 to create a transaction for an event


Step 1: Prepare for a transaction

You will need the "eventid" and "eventdateid" attributes to initiate a checkout. You can get these from a method such as /api/v2/events. Following sample inputs will be used in this example (eventid: 2138719424, eventdateid: 2138453919).

[GET] Prepare(api/v2/checkout/prepare)

Request: 

{{this.prepareinput|json}}

Response:

 {{this.events|json}}



Step 2: Create a transaction

POST values from the previous step's result object. The "tickettypes" object represents each registration spot or seat included in this transaction

[POST] Create(api/v2/checkout/create)


Request: 

{{this.datas|json}}

Response: 

 {{this.createorderresponse|json}}


Step 3: Fill the transaction with registration details

POST values from the previous step's result object. You will have to use the "checkout_id" property to populate this transaction and use the "tickets" object to populate the registration details including answers to your custom questions.

[POST] fillorder(api/v2/checkout/fillorder)

Request:

{{this.fill|json}}

Response:

{{this.fillorderresponse|json}}


Step 4: Confirm the transaction

POST values from the previous step's result object.

[POST] confirm(api/v2/checkout/confirm)

Request:

{{this.confirm|json}}

Response:

{{this.confirmorderresponse|json}}
